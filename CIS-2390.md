# CIS 2390 - Advanced Data Structures and Algorithms Analysis - 4 Credits

## Course Description
This course introduces students to the design, implementation, and evaluation of advanced data structures and algorithms. Topics include algorithmic efficiency, asymptotic analysis, recursion, trees, balanced trees, priority queues, heaps, hashing, maps, sets, sorting, graphs, and selected algorithm design techniques such as divide-and-conquer, greedy methods, backtracking, and dynamic programming. Students will analyze time and space requirements, compare alternative solutions, and apply data structures and algorithms to solve computational problems using a high-level programming language.

## Prerequisites
CIS 2353 - Data Structures

## Common Course Outcomes

- The student will be able to analyze algorithm efficiency using asymptotic notation, including Big-O, Big-Omega, and Big-Theta.
- The student will be able to compare data structure choices based on performance, memory use, implementation complexity, and suitability for a given problem.
- The student will be able to implement and use advanced data structures, including linked structures, stacks, queues, trees, balanced trees, heaps, maps, sets, and hash tables.
- The student will be able to represent and solve problems using graph data structures and graph algorithms, including problems involving relationships, networks, paths, and connectivity.
- The student will be able to design algorithmic solutions using strategies such as recursion, divide-and-conquer, greedy methods, backtracking, and dynamic programming.
- The student will be able to evaluate tradeoffs among competing algorithms by considering correctness, scalability, time complexity, space complexity, and practical implementation constraints.
- The student will be able to develop programming solutions that demonstrate appropriate use of data structures and algorithms, including testing, documenting, and explaining the reasoning behind the selected approach.
- Applies knowledge, concepts, paradigms, or theories to the assigned task ( GE – Critical Thinking )
