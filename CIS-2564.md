# CIS 2564 - Computer Organization, Architecture, and Assembly Language - 4 Credits

## Course Description
The architecture of computer systems and associated software. Topics include digital logic circuits, computer interfacing, interrupt systems, input/output systems, memory systems, assemblers, and assembly language programming.

## Prerequisites
CIS 2131 - Python Programming

## Common Course Outcomes

At the end of this course, students will be able to 
- Write, trace, and explain basic assembly language programs using a contemporary instruction set architecture.
- Explain the relationship between assembly language and machine code, including how instructions, operands, registers, and memory references are represented at the machine level.
- Represent common primitive data types in machine-level formats, including characters, integers, and floating-point values.
- Use registers, addresses, pointers, and memory access instructions to implement and manipulate basic data structures such as arrays and linked lists.
- Analyze and design basic combinational logic circuits using components such as logic gates, encoders, decoders, and multiplexers.
- Analyze and design basic sequential logic circuits using components such as flip-flops and registers.
- Explain how the CPU, registers, memory, and control signals interact during instruction execution.
- Describe register-level micro-operations, timing, and control mechanisms used in processor organization.
- Design simple register-level circuits that perform basic micro-operations.
- Explain the role of assemblers, memory systems, input/output, and interrupts in the organization and operation of computer systems.
- Applies knowledge, concepts, paradigms, or theories to the assigned task ( GE –
Critical Thinking )
