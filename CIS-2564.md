# CIS 2564 - Computer Organization, Architecture, and Assembly Language - 4 Credits

## Course Description
The architecture of computer systems and associated software. Topics include digital logic circuits, computer interfacing, interrupt systems, input/output systems, memory systems, assemblers, and assembly language programming.

## Prerequisites
CIS 2131 - Python Programming

## Common Course Outcomes

At the end of this course, students will be able to 
- Write, trace, and explain basic assembly language programs, including the relationship between assembly language, machine code, instructions, operands, registers, and memory.
- Represent and manipulate machine-level data using primitive data types, registers, addresses, pointers, memory access instructions, and basic data structures such as arrays and linked lists.
- Analyze and design basic combinational and sequential logic circuits using logic gates, encoders, decoders, multiplexers, flip-flops, and registers.
- Explain how the CPU, registers, memory, control signals, and instruction execution processes interact within a computer system.
- Describe register-level organization, including micro-operations, timing, and control mechanisms used in processor design.
- Design simple register-level circuits that perform basic micro-operations.
- Explain the roles of assemblers, memory systems, input/output subsystems, and interrupts in the organization and operation of computer systems.
- Applies knowledge, concepts, paradigms, or theories to the assigned task ( GE – Critical Thinking )
